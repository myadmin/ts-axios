# TypeScript study

学习 typescript，并实现 axios 浏览器部分，完善的 demo 与单元测试

## 脚手架
搭建项目脚手架，使用方法如下

```bash
git clone https://github.com/alexjoverm/typescript-library-starter.git YOURFOLDERNAME
cd YOURFOLDERNAME

# Run npm install and write your library name when asked. That's all!
npm install
```
